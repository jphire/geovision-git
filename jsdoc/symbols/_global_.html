deleteUntagged(bitscoreLimit)
Function for deleting nodes from the graph. If bitscoreLimit is not defined, all untagged nodes except root will be deleted. If bitscoreLimit is defined, all untagged nodes (except root) without connections over that bitscore will be deleted. It will also preserve a path to root from each remaining node (since many operations would break if graph became disconnected). Function will also clean all edges to (and from) the deleted nodes from the graph, and update data.bitscore field for remaining nodes. Will also untag all nodes once finished.
